DEPOSE TES IMAGES DE MARQUE ICI
================================

1) icon.png  -> Icone de l'application (celle affichee sur le telephone)
   - Format : PNG
   - Taille ideale : 1024 x 1024 pixels, carree
   - PAS de coins arrondis (le systeme les arrondit tout seul)
   - Fond de preference PLEIN (pas transparent) pour iOS

2) logo.png  -> Logo affiche DANS l'application (login, welcome, splash, etc.)
   - Format : PNG avec fond TRANSPARENT
   - Ton logo contient deja le texte du nom (choix "with_text")
   - Largeur conseillee : 600 a 1200 px, hauteur libre
   - Utilise un logo lisible en mode clair ET sombre, ou fournis 2 versions
     (logo.png = clair, logo_dark.png = sombre) et je gere l'adaptation.

Une fois les fichiers deposes ici, dis-le moi et je m'occupe du reste
(nom de l'app + generation des icones + branchement du logo).
